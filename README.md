# kanavmittal314.github.io

Personal website built with Jekyll and the Minimal Mistakes remote theme.

## Local Development

Use Ruby 3.3.4:

```sh
chruby-exec ruby-3.3.4 -- bundle install
chruby-exec ruby-3.3.4 -- bundle exec jekyll serve
```

The CS180 project writeups remain in the existing `1/`, `2/`, `3/`, `4/`, `5/`, and `final/` directories. The new `/cs180/` page links to them from the themed site navigation.
